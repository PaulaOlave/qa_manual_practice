Título:
El sistema entrega un mensaje confuso al ingresar credenciales inválidas.

Precondición: 
El usuario está en la página de login.

Pasos para reproducir:
1.Ingresar el nombre de usuario válido.
2.Ingresar la contraseña invalida.
3.Presionar el botón de login.

Resultado esperado:
El sistema entrega un mensaje claro respecto a la invalidez de la contraseña.

Resultado obtenido:
El sistema entrega un mensaje avisando nombre de usuario invalido.

Severidad: Media

Prioridad: Media
